# EMG_Adquisiton_Arduino_Unity
Desarrollo de un protocolo de entrenamiento pre-protésico innovador basado en el uso de un videojuego interactivo que, a través de la medición de la actividad electromiográfica de los músculos objetivos, permita evaluar la capacidad de personas con amputaciones trasradiales para adaptarse al uso de prótesis mioeléctricas.
